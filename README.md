# Optimisation Algorithms in Neural Networks

## 📘 Project Title
Comparative Study of Optimisation Algorithms in Neural Networks

---

## 🎯 Overview

This project investigates how different optimisation algorithms affect the training of neural networks. It focuses on three widely used methods:

- Stochastic Gradient Descent (SGD)
- RMSprop
- Adam (Adaptive Moment Estimation)

The study analyses how these algorithms influence convergence speed, stability, and overall model performance.

---

## 🧠 Objective

The objective of this project is to understand how optimisation strategies impact neural network training through both theoretical explanation and experimental evaluation.

---

## 🧪 Dataset

A synthetic dataset is generated using `make_moons`.

- 600 samples  
- Nonlinear structure  
- Suitable for analysing decision boundaries  

---

## ⚙️ Model Setup

- Neural network with two hidden layers (16 neurons each)  
- ReLU activation function  
- Same architecture used for all experiments  
- Only optimisation algorithm is changed  

---

## 📊 Results Summary

- SGD shows slower and less stable convergence  
- RMSprop improves stability through adaptive learning rates  
- Adam achieves the fastest convergence and best overall performance  

---

## 📈 Visualisations

The notebook includes:

- Training loss curves  
- Accuracy comparison  
- Decision boundary plots  

These visualisations demonstrate how optimisation algorithms affect training behaviour and model quality.
